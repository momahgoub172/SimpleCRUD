# SimpleCRUD
- This is ASP.NET Core Project
- Based on .Net 6
- Based on MVC Pattern

# Technologies Used:
- Asp.net (.Net 6)
- EF Core
- LINQ
- Asp.net Identity
- Sqlite as a Database

---------------------------
# Features:
## 1-CRUD:
- Create Movie / Genre
- Read Movie / Genre
- Update Movie / Genre
- Delete Movie / Genre

## 2-Other
- Activation Email using EmailSender Service

# Auth operations
- Login
- Logoff
- Register
- Reset Password
- Multi-Factor Authentication (using Key,QR Code)
